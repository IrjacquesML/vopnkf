🧪 Test du Menu Hamburger Mobile

Instructions :

  1. Ouvrez les outils de développement (F12)
  2. Activez le mode responsive (Ctrl+Shift+M ou Cmd+Shift+M)
  3. Sélectionnez un appareil mobile (iPhone, Samsung, etc.)
  4. Vous devriez voir le bouton hamburger (☰) en haut à droite
  5. Cliquez dessus pour ouvrir/fermer le menu

✅ Fonctionnalités à tester

  • Le bouton hamburger apparaît sur mobile (< 768px)
  • Le menu glisse depuis la droite quand on clique
  • Un overlay sombre apparaît derrière le menu
  • Le menu se ferme en cliquant sur l'overlay
  • Le menu se ferme en cliquant sur un lien
  • L'animation du bouton (☰ → ✕) fonctionne
  • Le scroll de la page est bloqué quand le menu est ouvert
  • La navbar se cache en scrollant vers le bas
  • La navbar réapparaît en scrollant vers le haut

📱 Résolutions à tester

  • Mobile : 375px × 667px (iPhone SE)
  • Mobile : 414px × 896px (iPhone 11 Pro Max)
  • Tablette : 768px × 1024px (iPad)
  • Desktop : 1920px × 1080px

📜 Contenu de test pour le scroll

Faites défiler cette page vers le bas pour tester le masquage automatique de la navbar sur mobile.
